Introducing… Mailprint Deluxe

Your eyes hurt from reading electronic mail on screen?

You prefer to have a stack of paper instead of a computer on your desk?

Do you want to see sender pictures in your printouts without senders setting the X-Face header?


UNIX™ wizards and a growing number of businesses rely on Mailprint Deluxe to print out their electronic mail!

Why power users pick Mailprint

Mailprint integrates well.

Mailprint is a classic UNIX™ tool and adheres to the UNIX™ pipe-and-filters architecture (mbox-in, groff-out).

Mailprint is a drop-in replacement for muttprint.

Mailprint is slim.

Mailprint is specifically engineered to be slim. While muttprint depends on the 250MB+ TeX Live distribution, Mailprint only depends on GNU roff, which is usually already installed to support the Linux man pages.

Mailprint is pretty.

Mailprint integrates with state-of-the-art profile picture sources to display sender pictures, such as X-Face and picon.

Mailprint is free.

Mailprint is Free Software and available free of charge at github.com/gnoack/mailprint.

Get started today!

What you need:

  • A UNIX™ or Linux™ workstation with GNU Groff.
    Groff is usually already installed on most Linux distributions.
  • Imagemagick's convert, if you want profile pictures.
  • The Go programming language, to install Mailprint from source.

To install Mailprint from source, run:

go install github.com/gnoack/mailprint/...@latest

Hook up mailprint with Mutt with the following Mutt configuration entry in .muttrc: 

set print_command="mailprint | lpr"
